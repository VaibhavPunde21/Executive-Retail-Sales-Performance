# 📊 Executive-Retail-Sales-Performance
Executive Retail Sales Performance using Power

## 🚀 Project Overview
This project focuses on analyzing sales performance using Power BI to generate actionable business insights.

## 🎯 Objectives
- Track key performance indicators (KPIs)
- Identify top-performing regions and products
- Analyze sales trends and patterns
- Support data-driven decision-making

## 🛠️ Tools Used
- Power BI
- Excel
- DAX

## 📊 Key Features
- Interactive dashboard with KPI cards
- Revenue, Profit, and Sales analysis
- Region-wise and category-wise performance
- Trend analysis using visual charts

## 🔍 Key Insights
- Identified top-performing regions contributing highest revenue
- Highlighted underperforming product categories
- Analyzed monthly sales trends for better forecasting
