# 📊 Executive Retail Sales Performance Dashboard

## 🚀 Project Overview
This project analyzes retail sales data to uncover key business insights and support data-driven decision-making.

The dashboard provides a comprehensive view of sales performance across regions, categories, and time periods.

---

## 🎯 Business Problem
Retail businesses often struggle to:
- Identify top-performing regions and products
- Track sales trends over time
- Detect underperforming segments

This project solves these challenges using interactive data visualization.

---

## 🛠️ Tools & Technologies
- Power BI  
- Excel  
- DAX  

---

## 📊 Dashboard Features
- KPI Cards (Revenue, Units Sold, AOV, Performance %)
- Region-wise sales analysis  
- Category performance insights  
- Time-based sales trends  
- Interactive filters for dynamic analysis  

---

## 🔍 Key Insights
- Identified top-performing regions contributing majority of revenue  
- Detected underperforming categories needing strategic focus  
- Observed seasonal sales trends impacting performance  
- Highlighted gaps between actual vs target sales  

---

## 💡 Business Recommendations
- Focus marketing efforts on high-performing regions  
- Improve strategy for low-performing categories  
- Optimize inventory based on sales trends  
- Use data-driven targets for better forecasting
